# Mercedes-AMG Petronas F1 — C-Suite Acceleration Platform

## Setup (One Time, 5 Minutes)

1. Create a GitHub account at github.com (if you don't have one)
2. Click "New Repository" → name it `mercedes-csuite` → Public → Create
3. Upload these 3 files to the repository:
   - `index.html`
   - `app.min.js`
   - `config.json`
4. Go to Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save
5. Wait 1-2 minutes → Your permanent URL: `https://YOUR-USERNAME.github.io/mercedes-csuite`
6. Bookmark this URL on every device

## How Updates Work

- **Mercedes/F1 data changes:** Tell Claude → Claude updates `config.json` → you copy-paste the new config.json content into GitHub → refresh your browser → done
- **New content (phases, simulations, flashcards):** Tell Claude → Claude gives you updated `app.min.js` → upload to GitHub → refresh → done
- **Your progress (tasks, quizzes, thesis):** Saved in YOUR browser automatically. Never lost.

## Files

| File | Size | Changes |
|------|------|---------|
| `index.html` | ~1KB | Never changes |
| `app.min.js` | ~290KB | Only when new features/content are added |
| `config.json` | ~1KB | When Mercedes numbers change (valuation, revenue, sponsors, etc.) |
